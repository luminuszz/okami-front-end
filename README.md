## Description

OKAMI FRONT-END is module for OKAMI consume OKAMI SERVER api rest


## Installation

For for run create a ```.env``` in de project dir

```bash
PORT=
NEXT_PUBLIC_API_URL=
API_URL=
DOCKER_FILE_ENV=

```

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start


# production mode
$ docker compose up --build
```
